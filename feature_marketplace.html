AI-Powered Marketplace

Intelligent E-Commerce for Agricultural Products

Feature Overview

The marketplace seamlessly integrates AI-powered product recommendations with diagnosis results, connecting farmers with the exact products they need. Sellers can register, list products, and manage orders, while buyers enjoy personalized recommendations based on their plant diagnoses. The system uses Egyptian currency (EGP) and supports local shipping addresses across all governorates.

AI-Powered Recommendations

The marketplace uses diagnosis results to provide contextual product recommendations:

Disease-to-Product Matching

// AI matches diagnosis to relevant products
const getRecommendedProducts = async (diagnosis: DiagnosisResult) => {
  const { diseaseName, severity, cropName } = diagnosis;
  
  // 1. Find products tagged with this disease
  const diseaseProducts = await findProductsByDisease(diseaseName);
  
  // 2. Prioritize by severity
  const products = severity === 'critical' 
    ? diseaseProducts.filter(p => p.category === 'chemical')
    : diseaseProducts;
  
  // 3. Apply AI ranking
  const rankedProducts = await rankByRelevance(products, {
    diseaseName,
    cropName,
    severity
  });
  
  return rankedProducts.slice(0, 6);  // Top 6 recommendations
};

Order Management

Order Lifecycle

1. Order Placement

Customer selects products, adds shipping address, places order with cash on delivery.

2. Seller Notification

Seller receives order notification via Firebase, reviews order details and inventory.

3. Order Confirmation

Seller confirms order, updates status to "Processing", prepares items for shipping.

4. Shipping

Order marked as "Shipped", tracking information provided to customer.

5. Delivery

Customer receives order, pays cash on delivery, confirms receipt.

6. Review & Rating

Customer can leave review and rating, seller receives feedback.

Access Restrictions

Guest User Limitations:

  • No Marketplace Access - Marketplace dropdown hidden from navigation
  • No Product Recommendations - AI recommendations not shown in scan results
  • Sign-In Encouragement - Beautiful cards prompting registration
  • Clear Value Proposition - Shows benefits of signing up

Authenticated User Benefits:

  • Full Marketplace Access - Browse, search, and purchase products
  • AI Recommendations - Contextual product suggestions after scans
  • Shopping Cart - Persistent cart across sessions
  • Order History - Track all purchases and deliveries
  • Seller Registration - Become a marketplace vendor
  • Reviews & Ratings - Leave feedback on products
