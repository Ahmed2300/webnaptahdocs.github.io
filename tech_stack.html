Technology Stack

Modern Web Technologies Powering Napta

Frontend Framework

React 18

Modern React with hooks, concurrent features, and automatic batching for optimal performance.

TypeScript

Type-safe development with interfaces, enums, and compile-time error checking.

Vite

Lightning-fast build tool with instant HMR and optimized production builds.

Tailwind CSS

Utility-first CSS framework for rapid UI development with custom theme configuration.

Backend & Infrastructure

Firebase Auth

Secure authentication with Google OAuth2 and session management.

Realtime Database

NoSQL cloud database with real-time synchronization across all clients.

Google Gemini AI

Advanced AI model for plant disease detection with 95% accuracy.

ImgBB API

Image hosting service with 32MB limit and automatic CDN distribution.
