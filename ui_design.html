Design Philosophy

Napta's design system is built on principles of clarity, accessibility, and natural aesthetics. The plant-themed color palette creates an organic feel while maintaining professional functionality. Every component is designed with user experience in mind, from the intuitive navigation to the satisfying micro-interactions.

Responsive Design

Breakpoint System

/* Tailwind CSS Breakpoints */
sm:  640px   /* Small devices (landscape phones) */
md:  768px   /* Medium devices (tablets) */
lg:  1024px  /* Large devices (desktops) */
xl:  1280px  /* Extra large devices (large desktops) */
2xl: 1536px  /* 2X large devices (larger desktops) */

/* Mobile-First Approach */
.container {
  width: 100%;           /* Mobile default */
  padding: 1rem;
}

@media (min-width: 768px) {
  .container {
    padding: 2rem;       /* Tablet and up */
  }
}

@media (min-width: 1024px) {
  .container {
    max-width: 1024px;   /* Desktop */
    margin: 0 auto;
  }
}
  • Mobile (< 768px) - Single column, bottom navigation, touch-optimized buttons
  • Tablet (768px - 1024px) - Adapted two-column, hybrid navigation
  • Desktop (> 1024px) - Full two-column layout, persistent top navigation

Accessibility Features

  • Semantic HTML - Proper heading hierarchy, landmarks, and ARIA labels
  • Keyboard Navigation - All interactive elements accessible via keyboard
  • Color Contrast - WCAG 2.1 AA compliant contrast ratios (4.5:1 minimum)
  • Focus Indicators - Clear visual focus states for navigation
  • Screen Reader Support - Descriptive alt text and ARIA attributes
  • Text Scaling - Content readable at 200% zoom level
  • Error Messages - Clear, actionable error descriptions
  • Touch Targets - Minimum 44x44px touch areas on mobile
